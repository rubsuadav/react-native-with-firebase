## Available Scripts

In the project directory, you can run:

For installing the dependencies:

```bash
npm install
```

For running the app on web:

```bash
npm run web
```

For running the app on android:

```bash
npm run android
```

For export the app to web:

```bash
npm run predeploy
```

For deploy the app to web:

```bash
npm run deploy
```

For testing the app:

```bash
npm run test
```
